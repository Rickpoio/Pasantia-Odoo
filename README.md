Proyecto Pasantia Odoo Developer

Construido con 
-Odoo v 19.0
-Postgres 17-bookworm
-Python 3.12.6

Requisitos para ejecutar el entorno:
-Tener Docker Desktop instalado 

Pasos para la ejecucion:
1. Clonar este repositorio y tenerlo en su entorno local
2. Crear un archivo .env en base al .env.example brindado para poder iniciar el proyecto
3. construiccion del entorno en el editor de texto que utilice usando el comando docker-compose up --build
4. desde el navegador en la barra de busqueda superior abrir el puerto local localhost:8069
